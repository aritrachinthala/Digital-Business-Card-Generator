# Digital-Business-Card-Generator
A simple frontend application built using HTML, CSS, and JavaScript.

## Features

- Enter personal details
- Upload profile image
- Add LinkedIn and GitHub links
- Choose from multiple card templates
- Generate digital business card
- Download as PNG image
- Download as PDF

## Technologies

- HTML
- CSS
- JavaScript
- html2canvas
- jsPDF

## How to Run

1. Download the project.
2. Open index.html in your browser.
3. Fill in your information.
4. Click Generate Card.
5. Download as Image or PDF.
